# הוראות הפעלת תכונת הדילוג

## ✅ מה שונה

### 1. **בלובי - המנהל יכול לבחור**
- יש כפתור toggle "אפשר לשחקנים לדלג"
- ברירת מחדל: מופעל (שחקנים יכולים לדלג)
- המנהל יכול לכבות את זה אם רוצה שכולם ישחקו

### 2. **במסך כתיבת האמת**
- אם המנהל אפשר דילוג, יש כפתור "דלג על התור שלי"
- שחקן שמדלג רואה "דילגת על התור!" ⏭️
- שחקן שכותב אמת רואה "האמת שלך נשמרה!" 🤫

### 3. **בלוגיקת המשחק**
- רק שחקנים שכתבו אמת (לא דילגו) מקבלים סיבוב
- אם כולם דילגו, המשחק עובר ישר לסיכום

---

## 🗄️ עדכון Database (חובה!)

**צריך להריץ את זה ב-Supabase SQL Editor:**

```sql
-- Add allow_skip column to games table
ALTER TABLE public.games 
ADD COLUMN IF NOT EXISTS allow_skip boolean NOT NULL DEFAULT true;
```

### איך להריץ:
1. לך ל-Supabase Dashboard
2. לחץ על "SQL Editor" בצד שמאל
3. לחץ "New query"
4. העתק את ה-SQL למעלה
5. לחץ "Run" (או Ctrl+Enter)

---

## 🧪 בדיקות

### מקרה 1: כולם משחקים (ברירת מחדל)
1. המנהל משאיר את "אפשר לשחקנים לדלג" מופעל
2. כל השחקנים כותבים אמת
3. ✅ המשחק ממשיך כרגיל

### מקרה 2: שחקן אחד מדלג
1. שחקן אחד לוחץ "דלג על התור שלי"
2. ✅ הסיבוב שלו לא מתרחש
3. ✅ עוברים לשחקן הבא

### מקרה 3: כולם מדלגים
1. כל השחקנים לוחצים "דלג על התור שלי"
2. ✅ המשחק עובר ישר לסיכום

### מקרה 4: המנהל אוסר דילוג
1. המנהל מכבה את "אפשר לשחקנים לדלג"
2. ✅ כפתור הדילוג לא מופיע במסך כתיבת האמת
3. ✅ כולם חייבים לכתוב אמת
